# Air-Fit Signal Engine

Internal GTM tool prototype that analyzes public company websites to identify signs of creative-operations pain relevant to Air.

## Planned flow

CSV of domains -> scrape -> extract signals -> analyze fit -> score -> review in dashboard/API

## Stack

- Python
- Playwright
- BeautifulSoup
- FastAPI
- Anthropic API
- Pandas
